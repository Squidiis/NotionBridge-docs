addSelectOption

Adds and selects a new option for a select or multi_select property of a Notion page.

Parameters

Name Type Description
pageId string ID of the Notion page to update.
propertyName string Name of the select or multi_select property.
newOption string The new option to add and select.
multiSelect boolean (Optional) Whether the property is a multi_select (default: true).

Example

const result = await notion.addSelectOption('pageId', 'propertyName', 'newOption', 'multiSelect');
        console.log(result);

Returns

A Promise resolving to the updated Notion page object.

Warning: This function relies on a Notion-side behavior (not a dedicated API endpoint) to add options.
Therefore, the select or multi_select property name must be spelled exactly as it exists on the page.
If the option does not already exist, Notion will automatically add it to the propertyâ€™s schema as a new choice.
